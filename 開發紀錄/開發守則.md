# 🛡️ XQ Dashboard 開發最高守則 (SOP)

## 📌 啟動自檢規則
1.  **讀取存檔**：啟動時必須優先讀取 `/開發紀錄/對話存檔.md`。
2.  **確認進度**：確認 `/開發紀錄/進度報告.md` 中的最後狀態。

## 🛠️ 排版與樣式守則 (重要！)
-   **雙端絕對隔離**：嚴禁直接修改全域 `th`, `td`, `tr` 的寬度。所有佈局變動必須分別寫在 `@media (min-width: 601px)` (電腦) 與 `@media (max-width: 600px)` (手機) 內。
-   **手機標題防禦**：手機版 Header 空間極度敏感，嚴禁在未經空間計算的情況下增加 Header 元素，且必須鎖定 `height` 與 `overflow`。
-   **卡片獨立性**：手機版必須維持「獨立浮動卡片」樣式，外層容器 (`.table-container`) 必須保持透明無邊框。

## 🔄 自動化與同步守則
-   **監控必開**：確保 `啟動全自動監控.bat` 執行中。
-   **即時紀錄**：任何代碼變動後，必須同步更新「進度報告」與「對話存檔」。
