# XQ 智慧監控清單 - 開發進度報告

本專案旨在將 XQ 匯出的籌碼資料自動化轉化為專業的 Web 儀表板，支援電腦與手機雙端同步監控。

## 📅 最後更新日期：2026-05-08 (22:22 更新)

---

## ✅ 已完成核心功能

### 1. 資料處理與環境防錯
- **一鍵同步 (`啟動網頁.bat`)**：整合了 Excel 資料轉換、Git 上傳、GitHub Pages 自動部署。
- **環境智慧偵測**：啟動時自動檢查 Node.js 與 Git，若未安裝會自動跳出下載提示，方便換電腦使用。
- **編碼優化**：自動處理 Big5 編碼，解決 Excel 匯出亂碼問題。

### 2. 頂級視覺與 UI/UX 優化
- **iPhone 毛玻璃質感 (Glassmorphism)**：自選股模式下呈現極具現代感的磨砂玻璃懸浮特效。
- **標題與佈局分流**：電腦版維持專業大氣，手機版自動轉為易讀的卡片模式。
- **動態回饋**：愛心收藏點擊時有跳動動畫，讓操作更具有真實感。
- **數字標籤 (Badge)**：自選按鈕與底部選單會即時顯示目前收藏的股票總數。

### 3. 選股監控邏輯
- **優先置頂模式**：連買 3 週以上的股票自動標記「閃電符號」並優先置頂。
- **自選股收藏系統**：資料持久化儲存於本地瀏覽器，支援全裝置收藏操作。

## 🛠 維護說明
1. **更新資料**：將 XQ 匯出的最新 Excel (CSV) 放入專案根目錄，點擊執行 `啟動網頁.bat` 即可。
2. **換電腦銜接**：直接複製整個資料夾，確保新電腦安裝了 Node.js 與 Git 即可啟動。
3. **自選同步說明**：自選股目前儲存在裝置瀏覽器中（電腦與手機收藏目前不互通）。
